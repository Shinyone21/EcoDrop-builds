# EcoDrop
学校の研究の一環で作成した、2D思考型アクションゲーム「EcoDrop」の公開ページです。

※このリポジトリにはUnityプロジェクト本体は含まれておりません。</br>
ライセンスの関係により、素材そのものを私の独断で公開することが難しいためです。</br>
ソースコードには、GitHub Pages用のUnity WebGLビルドが含まれています。

## Game Overview

上から落ちてくるゴミをプレイヤーが操作して、
正しいゴミ箱へ導く2Dアクションゲームです。

日常にありふれるゴミの問題を身近に感じ、ゴミ分別をより意識して行なって貰うことを目指した開発中のゲームです。


## How To Play

キーボードでは、← →の矢印キーか、A Dキーで、タッチパネルでは、画面の右側と左側をタップすることで、操作が可能です。

また、Xキーで、ゲーム中メニューが開く、Rでリトライ、Tでタイトル推移などのキーボードショートカットがあります。
