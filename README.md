# EcoDrop

学校の総合探究の一環として制作しました。

※このリポジトリにはUnityプロジェクト本体は含まれていません。  
一部アセットのライセンスの都合で再配布できないため、  
GitHub Pages 用の WebGL ビルドのみ公開しています。

## ゲーム概要

このゲームは、落ちてくるゴミを左右に操作して  
正しいゴミ箱に入れる2Dアクションゲームです。

ゲームを通して、日常生活でのゴミ分別や  
環境問題への意識を少しでも持ってもらうことを目的としています。

## 操作方法

キーボード：
- 矢印キー ← →
- A / D キー

タッチ操作：
- 画面の左半分 / 右半分をタップ

ショートカットキー：
- X : ゲーム内メニュー
- R : リトライ
- T : タイトルに戻る
