created as part of a school research project.

## Game Overview

This is a 2D action game where the player controls falling trash
and guides it into the correct trash bin.

The goal of this game is to help players become more aware of
waste sorting and environmental issues in everyday life.

## How To Play

Keyboard:
- Arrow keys ← →
- A / D keys

Touch screen:
- Tap left or right side of the screen

Keyboard shortcuts:
- X : Open in-game menu
- R : Retry
- T : Return to title
